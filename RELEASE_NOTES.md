# Markdown Editor Electron 项目发版说明

## 版本 1.0.0

### 新功能
- 支持 Markdown 文件的创建、打开、保存和另存为功能。
- 支持 Markdown 文件的实时预览。
- 支持插入代码块和图片。
- 支持从剪贴板粘贴图片。
- 支持导出为 PDF 文件。
- 支持大纲视图，显示文档结构。
- 支持自动保存功能。

### 优化
- 优化编辑区和预览区的样式，使其更加美观。
- 优化代码块和图片的渲染规则。
- 优化大纲栏的显示和隐藏功能。
- 优化窗口状态的保存和恢复功能。

### 修复
- 修复了在打包时遇到的权限问题。
- 修复了菜单项状态更新的问题。

### 依赖更新
- 更新 `electron` 到版本 `^33.3.1`。
- 更新 `electron-builder` 到版本 `^25.1.8`。
- 更新 `highlight.js` 到版本 `^11.11.1`。
- 更新 `markdown-it` 到版本 `^14.1.0`。

### 其他
- 添加 Apache 2.0 许可证。
- 添加 `.gitignore` 文件，忽略 `node_modules` 和 `dist` 目录。
