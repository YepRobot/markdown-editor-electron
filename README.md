# Markdown Editor

一个基于 Electron 的现代化 Markdown 编辑器。

## 功能特点

- 实时预览 Markdown 内容
- 支持代码块语法高亮
- 支持图片拖拽和粘贴
- 支持导出 PDF
- 支持大纲导航
- 自动保存功能

## 开发环境要求

- Node.js >= 14.0.0
- npm >= 6.0.0

## 安装与运行

```bash
# 安装依赖
npm install

# 启动应用
npm start

# 打包应用
npm run package
```

## 使用说明

1. 使用 `Ctrl+N` 创建新文件
2. 使用 `Ctrl+S` 保存文件
3. 使用 `Ctrl+O` 打开文件
4. 使用 `Ctrl+E` 导出为 PDF
5. 使用 `Ctrl+T` 切换大纲视图
